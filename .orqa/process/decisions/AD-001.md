## Decision

Rust owns the domain model — sessions, artifacts, governance state, agent orchestration. Svelte is a view layer that renders what Rust tells it.

## Rationale

The domain logic (session management, artifact parsing, Claude API integration, file system tools) is complex and benefits from Rust's type system, performance, and reliability. Keeping it in the backend means the frontend stays simple and testable.

## Consequences

All business logic lives in `backend/src-tauri/src/domain/`. The frontend never makes decisions about data flow or state management beyond UI concerns.
